Hospital Management API
A simple Hospital Management API building using FastAPI. This project allows us to create, retrive , retrive by sorting & id, update & delete
patients , doctors  and appointment entities, using JSON based storage

CRUD operation
Data validation with Pydantic
Storage in JSON
Modular routing

Features :

Patient
  Create patient
  View all patients
  View a specific patient
  Update patient details
  Delete patient
  Sort patients by weight, height, or BMI

Doctor
  Create Doctor
  View all doctor
  View a specific doctor
  Update doctor detail
  Delete doctor

Appointment
  Create appointment between doctors and patients
  Validating existience of patient and doctor
  Store appointment schedual with date and time
